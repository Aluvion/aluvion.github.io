安装

因为我已经安装过 vue 了，所以直接使用 vue create 一个新的项目。

安装 Electron 来结合 Vue 网上有各种各样的说法，我采用的是创建完 Vue 项目后再安装 vue-cli-plugin-electron-builder 的方式：https://www.jianshu.com/p/dfcf2a6a497c 。
