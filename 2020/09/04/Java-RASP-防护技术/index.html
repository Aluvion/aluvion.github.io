Java RASP 防护技术

前言

久闻 OpenRASP 大名，对其 Java 防护方式好奇，所以研究一下 RASP，看看它是怎么实现命令执行拦截的。

尝试

尝试用 javassist 修改 Runtime 类，结果发现主要会有两个问题：

  • 类加载器无法覆盖启动类加载器加载的类
  • 同个 Class 不能在同个类加载器中重复加载，要修改一个类就需要创建一个新的类加载器

不会写，只能先看看别人的实现了。
