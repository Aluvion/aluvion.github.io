简单来说就是在 SQL 查询之间插入的 SQL 操作，而建立连接时会执行的 set autocommit=1 等操作正好可以触发，所以如果可以控制 JDBC 连接串，就可以通过设置 queryInterceptors 的方式让客户端反序列化我们返回的数据，具体危害程度由客户端的环境决定。
